# 🚀 Curso GitHub Demo - Proyecto Python

## 📌 Descripción
Este repositorio fue creado como parte de la Evalucaión final trabajo práctico del módulo DevOps & Project Management Fundamentals del Diploma Devops Enginneer de DMC. El objetivo es  demostrar el uso de GitHub con buenas prácticas en el manejo de repositorios, ramas, commits y pull requests.

El proyecto incluye un script simple en Python que muestra un mensaje de bienvenida.

---

## 🧱 Estructura del proyecto
```bash
curso-github-demo/
├── app.py
├── README.md
└── .gitignore
```

---

## 🌿 Flujo de trabajo Git

Se trabajó con las siguientes ramas:
- `main`: rama principal
- `develop`: rama de integración de desarrollo
- `feature/python-script`: desarrollo de la funcionalidad

### Flujo aplicado:

1. Creación de rama feature desde `develop`
2. Desarrollo de script simple en Python
3. Realización de commits con buenas prácticas
4. Creación de Pull Request hacia `develop`
5. Revisión y aprobación de pull request
6. Levantamiento de observaciones de pull request
7. Merge hacia `develop`
8. Merge final hacia `main`

---

## 💻 Ejecución del proyecto

Debes de tener Python instalado y ejecutar:

```bash
python app.py

```

## 🧪 Buenas prácticas aplicadas
